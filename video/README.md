# Директория для видеофайлов

Эта директория предназначена для хранения видеофайлов, которые будут воспроизводиться ботами в видеочатах Telegram.

## Поддерживаемые форматы

- `.mp4` - MP4 видео (рекомендуется)
- `.avi` - AVI видео
- `.mkv` - Matroska видео
- `.mov` - QuickTime видео
- `.webm` - WebM видео

## Использование

1. Поместите ваши видеофайлы в эту директорию
2. Боты автоматически обнаружат их при запуске функции "Голосовой чат"
3. Видео будут воспроизводиться вместе с аудио из директории `audio/`

## Примечания

- Убедитесь, что у вас установлен `ffmpeg` для обработки видео
- Видеофайлы должны иметь подходящее качество и разрешение для трансляции
- Рекомендуется использовать видео с соотношением сторон 16:9 или 4:3
- Для лучшей производительности используйте сжатие H.264 для видео

## Требования

Для работы с видео необходимо:
- `pytgcalls` - библиотека для работы с голосовыми/видео чатами Telegram
- `ffmpeg` - инструмент для обработки медиафайлов

Установка:
```bash
pip install pytgcalls
# Для macOS:
brew install ffmpeg
# Для Ubuntu/Debian:
sudo apt-get install ffmpeg
```
