# Директория для аудиофайлов

Эта директория предназначена для хранения аудиофайлов, которые будут воспроизводиться ботами в голосовых чатах Telegram.

## Поддерживаемые форматы

- `.mp3` - MP3 аудио (рекомендуется)
- `.wav` - WAV аудио
- `.ogg` - OGG Vorbis аудио
- `.m4a` - M4A/AAC аудио
- `.flac` - FLAC аудио

## Использование

1. Поместите ваши аудиофайлы в эту директорию
2. Боты автоматически обнаружат их при запуске функции "Голосовой чат"
3. Аудио будут воспроизводиться по очереди или в случайном порядке (в зависимости от настроек)

## Текущие файлы

В директории уже находятся примеры аудиофайлов:
- `ahhh-for-donbass.mp3`
- `from-screw-bass-bossted.mp3`
- `the-sound-of-starting-windows-at-full-volume-300-take-care-of-your-ears.mp3`

## Примечания

- Рекомендуется использовать файлы с битрейтом 128-320 kbps для хорошего качества
- Файлы будут воспроизводиться в алфавитном порядке (режим sequential) или случайно (режим random)
- Все файлы из директории будут использоваться автоматически

## Требования

Для работы с аудио необходимо:
- `pytgcalls` - библиотека для работы с голосовыми чатами Telegram
- `ffmpeg` - инструмент для обработки медиафайлов

Установка:
```bash
pip install pytgcalls
# Для macOS:
brew install ffmpeg
# Для Ubuntu/Debian:
sudo apt-get install ffmpeg
```
